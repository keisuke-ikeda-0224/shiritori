# しりとり vs AI

ブラウザで遊べる日本語しりとりゲームです。AIと対戦しましょう！

## 遊び方

1. `index.html` をブラウザで開く
2. 表示された文字から始まるひらがなの言葉を入力して「送る」を押す
3. 制限時間は **10秒**
4. 「ん」で終わる言葉を言ったら負け
5. AIが言葉を思いつかなければあなたの勝ち！

## ルール

- ひらがなのみ入力可能
- 一度使った言葉は再利用不可
- 「ん」で終わったら即負け
- タイムオーバーもその番のプレイヤーの負け

## 技術

HTML / CSS / JavaScript のみで動作。外部ライブラリ不要。
